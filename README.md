# Excel-Coffee-Sales

This project is a clean and practical Excel-based analysis of coffee sales data.

## Project Files
- <a href="https://github.com/anthonymike180/Excel-Coffee-Sales/blob/main/coffeeOrdersData.xlsx">Dataset</a>
- <a href="https://github.com/anthonymike180/Excel-Coffee-Sales/blob/main/coffeeOrdersProject.xlsx">Dashboard</a>

##  Objectives  
- Analyze monthly sales trends for the coffee business.  
- Identify top-selling products and customer segments.  
- Build visual dashboards in Excel to track revenue, units sold, and regional performance.  
- Provide actionable insights to help the business grow.

##  Key Features  
- Pivot tables and charts that summarize sales by product, region and time.  
- Conditional formatting to highlight high-/low-performing sales.  
- Dashboard sheet that gives a clear overview of KPIs: total revenue, units sold, average sale value.
